## [1.0.0] - 2025-09-21

### Added

- Initial release of the Remnote Comments Plugin.
- Features:
    - Add, edit, and remove comments in any Rem.
    - View all comments in a sidebar.
    - Click to navigate to the relevant Rem.
    - Beautiful Modern UI with animations
